# 简述
简单工厂模式并非GoF的23种设计模式之一。它是工厂模式家族中最简单使用的模式，可以理解为不同工厂模式的一个特殊实现

# 案例

- 以汽车的生产为例，不同厂家生产的汽车即为不同的产品，但它们都拥有汽车共同的属性，因此设定汽车的基类ICar，不同品牌的汽车继承ICar。
- 生产汽车需要有工厂，因此设定类Factory根据不同的名字来生产不同型号的汽车。
